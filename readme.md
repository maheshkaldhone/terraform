# 🚀 Terraform AWS Practice Repository

This repository contains reusable Terraform modules and sample infrastructure setups for practicing AWS resource provisioning using Infrastructure as Code (IaC).

## 📌 Objectives

- Learn and practice Terraform with real AWS services
- Build modular, reusable, and environment-specific infrastructure
- Follow best practices in code structure and tagging
- Experiment with multiple AWS services including EC2, S3, VPC, RDS, and more
